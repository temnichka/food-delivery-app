Food Delivery App — Приложение для доставки еды

Описание проекта
Сервис для заказа и доставки еды из ресторанов. Включает мобильное приложение для клиентов, веб-сайт для администраторов и серверное API.

Структура репозитория

 Папка           Назначение 
 MobileApp       Мобильное приложение — клиентская часть для пользователей 
 Web             Веб-интерфейс — панель администратора и партнёров 
 ServerApi       Серверная часть — REST API, бизнес-логика, БД
